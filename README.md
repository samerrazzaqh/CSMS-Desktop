## 📌 Description  

This project is a hands-on **C# Form** implementation of an CSMS store, covering key concepts such as:  

- Building the **UI** entirely using **Form Desktop **.  
- **Integration** with **Authentication API** to understand **Guards & Layouts**.  
- **Integration** with an API to display various data, including:  
  - **Products**  
  - **Product Details**  
  - **Categories**
